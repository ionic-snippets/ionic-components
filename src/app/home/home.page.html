 📋 Component Properties

Inputs:

  • rating: Number of selected stars
  • maxRating: Maximum number of stars
  • iconType: Icon type (star, heart, thumbs-up, diamond, square, circle)
  • editable: Whether editing is allowed or read-only
  • size: Size (small, medium, large)
  • color: Color of filled stars
  • emptyColor: Color of empty stars
  • showRating: Show rating text

Outputs:

  • ratingChange: Emitted when rating changes
  • ratingClick: Emitted when a star is clicked
